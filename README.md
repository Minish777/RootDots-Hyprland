# 🌲 Into the Woods

Мой личный «лесной» райс для **Hyprland**.

---

### 🛠 Компоненты
* **WM**: Hyprland.
* **Shell**: Fish + Starship.
* **Terminal**: Kitty.
* **Bar**: Waybar
* **Menu**: Fuzzel .
* **Visuals**: Cava и awww.

### ⌨️ Хоткеи
* `SUPER + Enter` — Вызов терминала (Kitty).
* `SUPER + R` — Лаунчер приложений (Fuzzel).
* `SUPER + V` — Менеджер буфера обмена (Cliphist).
* `SUPER + M` — Меню выхода/питания.
* `SUPER + Shift + R` — Мгновенный релоад всех конфигов.
